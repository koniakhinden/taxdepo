# Резервная копия версии v1 (до редизайна)

Здесь лежит исходный код рабочей версии до перехода на дизайн «финансового кабинета».
Эти файлы НЕ участвуют в сборке (Next.js компилирует только корневой /app).

## Как откатиться
Вариант 1 (быстрее всего): на Vercel → Deployments → выбрать деплой рабочей версии → «Promote to Production» (мгновенный откат, ничего не пересобирая).

Вариант 2 (через код): скопировать содержимое этой папки обратно в корень, заменив app/ и lib/, затем git commit + push.
